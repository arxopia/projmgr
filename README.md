ProjMgr
===

ProjMgr is a simple source code project manager for updating and checking local changes on multiple projects at once.

Requirements
---

* ruby 
* rubygems
* choice
* yaml 


Installation
---
Installation is really easy just gem install!

	% sudo gem install projmgr
